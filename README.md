# PowerShell Comprobacion Integridad SHA512 -  V.2.1.0
Script PowerShell para generar y verificar manifiestos de integridad de archivos y carpetas usando SHA512. Detecta archivos modificados, eliminados, movidos o nuevos. Incluye protección automática del manifiesto y menús interactivos fáciles de usar. Ideal para respaldos y seguridad de datos.
